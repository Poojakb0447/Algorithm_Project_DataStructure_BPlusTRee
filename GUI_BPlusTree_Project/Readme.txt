B+ Tree Application
----------------------------------------------------

This README file explains how to use the B+ Tree Application program which is developed
using Java language.

The B+ Tree Application program includes Java source code files for logic and user interface. The users are able to load an input file into a B+ Tree and insert, delete,update and search for a node from thr tree. The changes can be saved in the input file. 

Steps to run the program:

1) To execute the program, open the program in Eclipse.
2) Right click on the name of the program in the Project Explorer, select Run As --> Java Application.
